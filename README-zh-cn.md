* Microsoft PowerPoint  
* Microsoft Publisher  
* Microsoft Skype 企业版  
* Microsoft Word  

## 其他应用  
* Microsoft Project  
* Microsoft Visio  

## 安装  
* **支持的操作系统：** Windows 10  
  * 构建版本 1703 或更高  
* **前提条件：**  
  * 卸载所有已安装的 MSI 版本 Office：  
    * Microsoft Office 2019 及更早版本  
    * Microsoft Office 365  
  * 网络连接：  
    * 有线网络（推荐）  
    * 无线网络  
  * 管理员权限  

## 激活  
* **需要网络连接：**  
  * 有线网络（推荐）  
  * 无线网络  
* **需要有效的产品密钥：**  
  * 安装前将密钥添加到 `configuration.xml` 文件  
  * 或在安装后使用 **更改产品密钥**
