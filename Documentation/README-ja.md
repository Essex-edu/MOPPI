## インストール  
* **対応OS:** Windows 10  
  * OSビルド 1703 以降  
* **必要条件:**  
  * 既存のMicrosoft Office MSI版の削除:  
    * Microsoft Office 2019以前  
    * Microsoft Office 365  
  * インターネット接続:  
    * 有線LAN（推奨）  
    * Wi-Fi  
  * 管理者権限  

## ライセンス認証  
* **インターネット接続が必要:**  
  * 有線LAN（推奨）  
  * Wi-Fi  
* **有効なプロダクトキーが必要:**  
  * インストール前に `configuration.xml` にキーを追加  
  * またはインストール後に **プロダクトキーの変更** を使用
